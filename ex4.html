Ex.4 : HTML : BEGINNER : สร้างหน้าโปรไฟล์ส่วนตัวง่ายๆ ด้วย HTML

คุณได้รับมอบหมายให้สร้างหน้าเว็บโปรไฟล์ส่วนตัวอย่างง่ายโดยใช้ HTML พื้นฐาน เพื่อแสดงข้อมูลเกี่ยวกับตัวคุณเอง จุดประสงค์คือเพื่อให้คุณคุ้นเคยกับการใช้แท็ก HTML ทั่วไป เช่น หัวข้อ, ย่อหน้า, รูปภาพ, รายการ และลิงก์

สิ่งที่โจทย์ต้องการ :

  • หน้าเว็บต้องมีชื่อเรื่อง (title) ในแท็ก <head> ที่เหมาะสมกับโปรไฟล์ส่วนตัว (เช่น "โปรไฟล์ของ [ชื่อของคุณ]")
  • ต้องมีส่วนหัวหลัก (<h1>) ที่แสดงชื่อของคุณ
  • ต้องมีรูปภาพโปรไฟล์ (<img>) สามารถใช้รูป placeholder ได้ เช่น https://via.placeholder.com/150
  • ต้องมีส่วนแนะนำตัวสั้นๆ หรือคำอธิบายเกี่ยวกับตัวคุณ (ใช้ <p>)
  • ต้องมีรายการข้อมูลที่น่าสนใจเกี่ยวกับตัวคุณ เช่น งานอดิเรก หรือสิ่งที่ชอบ (ใช้ <ul> และ <li> อย่างน้อย 3 รายการ)
  • ต้องมีลิงก์ไปยังโซเชียลมีเดียหรือเว็บไซต์อื่นๆ (ใช้ <a>) อย่างน้อย 2 ลิงก์
