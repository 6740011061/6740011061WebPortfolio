Ex.6 : HTML : BEGINNER : สร้างหน้าโปรไฟล์ผู้ใช้แบบ Responsive ด้วย HTML

คุณได้รับมอบหมายให้สร้างหน้าโปรไฟล์ผู้ใช้โดยใช้เฉพาะ HTML เท่านั้น เป้าหมายคือการจัดโครงสร้างเนื้อหาโปรไฟล์ให้มีความหมายและรองรับการแสดงผลบนอุปกรณ์ที่หลากหลาย (responsive-ready) โดยเน้นการใช้แท็ก HTML เชิงความหมาย (semantic HTML) เพื่อปรับปรุงการเข้าถึง (accessibility) และความสามารถในการค้นหา (SEO) ของหน้าเว็บนี้ หน้าโปรไฟล์ควรประกอบด้วยส่วนหลักๆ เช่น ข้อมูลส่วนตัว, ประวัติย่อ, ทักษะ และลิงก์ไปยังโซเชียลมีเดียต่างๆ

สิ่งที่โจทย์ต้องการ :

  • สร้างโครงสร้างพื้นฐานของหน้าเว็บด้วย <!DOCTYPE html>, <html>, <head>, และ <body> ใช้แท็ก meta viewport เพื่อรองรับการแสดงผลแบบ responsive
  • สร้างส่วนหัว (header) สำหรับชื่อโปรไฟล์หรือชื่อเว็บไซต์ สร้างส่วนหลัก (main content) ที่รวมข้อมูลโปรไฟล์ทั้งหมด
  • ใช้แท็ก HTML เชิงความหมาย (semantic HTML) เช่น <header>, <nav>, <main>, <section>, <article>, <aside>, <footer> เพื่อแบ่งส่วนเนื้อหาให้ชัดเจน
  • แสดงรูปภาพโปรไฟล์โดยใช้แท็ก <img> พร้อมแอตทริบิวต์ alt ที่เหมาะสม
  • แสดงชื่อผู้ใช้และอาชีพ/ตำแหน่ง โดยใช้แท็กหัวข้อ (heading tags) ที่เหมาะสม (เช่น <h1>, <h2>)
  • มีส่วนสำหรับประวัติโดยย่อหรือเกี่ยวกับฉัน (About Me) โดยใช้แท็ก <h2> และ <p>
  • มีส่วนสำหรับแสดงทักษะ (Skills) โดยใช้รายการแบบไม่เรียงลำดับ (unordered list <ul>) และรายการ (list item <li>)
  • มีส่วนสำหรับลิงก์โซเชียลมีเดีย (Social Media Links) โดยใช้แท็ก <a> และอาจจะใช้รายการแบบไม่เรียงลำดับเพื่อจัดกลุ่ม
