Ex.1 : CSS : BEGINNER : สร้างการ์ดสินค้าด้วย CSS

โจทย์นี้จะช่วยให้คุณฝึกฝนการใช้งาน CSS พื้นฐานเพื่อสร้างการ์ดสำหรับแสดงข้อมูลสินค้า คุณจะต้องจัดรูปแบบการ์ดที่กำหนดให้ดูดี อ่านง่าย และน่าสนใจ โดยใช้คุณสมบัติ CSS ต่างๆ เช่น สีพื้นหลัง, การจัดกึ่งกลาง, ระยะห่าง, ขอบมน และเงา

สิ่งที่โจทย์ต้องการ:

  • ทำให้ .product-card มีพื้นหลังสีขาว (#FFFFFF)
  • เพิ่ม padding ให้กับการ์ดอย่างน้อย 20px
  • เพิ่ม border-radius ให้กับการ์ด (เช่น 10px) เพื่อความสวยงาม
  • เพิ่ม box-shadow ให้กับการ์ดเพื่อให้ดูเหมือนลอยอยู่ (เช่น 0 4px 8px rgba(0,0,0,0.1))
  • จัดเนื้อหาทั้งหมดภายใน .product-card ให้อยู่กึ่งกลางในแนวนอน (text-align: center;)
  • ตั้งค่า max-width ของการ์ดเป็น 300px และ margin: auto; เพื่อจัดกึ่งกลางการ์ดและจำกัดความกว้าง
  • จัดรูปแบบ img ภายใน .product-card ให้มี max-width: 100%; และ height: auto;
  • จัดรูปแบบ .product-title ให้เป็นตัวหนา (font-weight: bold;) และมีขนาดใหญ่ขึ้น (เช่น font-size: 1.5em;)
  • จัดรูปแบบ .product-price span ให้มีสีที่โดดเด่นขึ้น เช่น สีเขียว (#28a745)
  • จัดรูปแบบปุ่ม .add-to-cart-btn ให้มีพื้นหลังสีน้ำเงิน (#007bff) ข้อความสีขาว (#FFF) และมีขอบมน (เช่น border-radius: 5px;) รวมถึง padding ที่เหมาะสม
