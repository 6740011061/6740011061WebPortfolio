Ex.8 : CSS : Intermediate : สร้าง Responsive Product Card Layout

คุณได้รับมอบหมายให้สร้างหน้าเว็บสำหรับแสดงรายการสินค้า โดยแต่ละสินค้าจะแสดงอยู่ในรูปแบบของการ์ด (Product Card) ที่สวยงามและสามารถปรับเปลี่ยนรูปแบบการแสดงผลได้ตามขนาดหน้าจอ (Responsive Design) เพื่อให้ผู้ใช้ได้รับประสบการณ์ที่ดีที่สุดไม่ว่าจะเข้าชมจากอุปกรณ์ใดก็ตาม

สิ่งที่โจทย์ต้องการ :

  • ใช้ Flexbox หรือ CSS Grid ในการจัดเรียงการ์ดสินค้าในคลาส product-grid
  • จัดสไตล์ให้กับ product-card ให้มีการแสดงผลที่สวยงาม เช่น มีขอบมน (border-radius), เงา (box-shadow), ระยะห่างภายใน (padding) และระยะห่างภายนอก (margin)
  • จัดสไตล์ให้กับรูปภาพสินค้า (img) และข้อมูลสินค้า (ชื่อ, ราคา, ปุ่ม) ภายใน product-card ให้เหมาะสม
  • ใช้ Media Queries เพื่อทำให้การ์ดสินค้าเป็น Responsive โดยมีเงื่อนไขดังนี้:
    • บนหน้าจอขนาดใหญ่ (เช่น 1024px ขึ้นไป) แสดง 3-4 การ์ดต่อแถว
    • บนหน้าจอขนาดกลาง (เช่น 768px - 1023px) แสดง 2 การ์ดต่อแถว
    • บนหน้าจอขนาดเล็ก (เช่น ต่ำกว่า 768px) แสดง 1 การ์ดต่อแถว
