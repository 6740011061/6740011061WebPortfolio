Ex.7 : JavaScript : BEGINNER : สร้างตัวนับเลขง่ายๆ

ในโจทย์นี้ คุณจะต้องสร้างหน้าเว็บที่มีตัวนับเลขที่สามารถเพิ่มค่าและลดค่าได้ด้วยการคลิกปุ่ม คุณจะต้องใช้ JavaScript เพื่อจัดการกับการโต้ตอบของผู้ใช้และอัปเดตค่าที่แสดงบนหน้าจอ

สิ่งที่โจทย์ต้องการ :

  • สร้างตัวแปร JavaScript สำหรับเก็บค่าปัจจุบันของตัวนับ (เริ่มต้นที่ 0)
  • แสดงค่าปัจจุบันของตัวนับบนหน้าเว็บ (element ที่มี id เป็น count-display)
  • เมื่อคลิกปุ่ม "เพิ่มค่า" (element ที่มี id เป็น increment-btn) ค่าของตัวนับควรเพิ่มขึ้นทีละ 1 และแสดงผลบนหน้าจอ
  • เมื่อคลิกปุ่ม "ลดค่า" (element ที่มี id เป็น decrement-btn) ค่าของตัวนับควรลดลงทีละ 1 และแสดงผลบนหน้าจอ
