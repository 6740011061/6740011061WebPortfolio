Ex.3 : CSS : Intermediate : สร้างการ์ดสินค้าแบบตอบสนองพร้อมเอฟเฟกต์โต้ตอบ

คุณได้รับมอบหมายให้สร้างชุดการ์ดสินค้า (Product Cards) ที่สวยงามและตอบสนอง (Responsive) ต่อขนาดหน้าจอต่างๆ โดยใช้ HTML และ CSS คุณจะต้องออกแบบการ์ดที่แสดงรูปภาพ ชื่อสินค้า รายละเอียดสั้นๆ และปุ่ม 'ดูรายละเอียด' นอกจากนี้ การ์ดควรมีเอฟเฟกต์โต้ตอบเมื่อผู้ใช้นำเมาส์ไปชี้ (hover) และใช้ CSS Custom Properties เพื่อจัดการสีหลักของธีม

สิ่งที่โจทย์ต้องการ:

  • สร้างโครงสร้าง HTML สำหรับการ์ดสินค้าแต่ละใบ รวมถึงรูปภาพ, หัวข้อ, คำอธิบาย และปุ่ม 'ดูรายละเอียด'
  • จัดเรียงการ์ดในรูปแบบกริด (Grid Layout) ที่ตอบสนองต่อขนาดหน้าจอ: แสดง 1 คอลัมน์บนหน้าจอขนาดเล็ก (เช่น มือถือ), 2 คอลัมน์บนหน้าจอขนาดกลาง (เช่น แท็บเล็ต) และ 3 คอลัมน์ขึ้นไปบนหน้าจอขนาดใหญ่ (เช่น เดสก์ท็อป) โดยใช้ Media Queries
  • เพิ่มเอฟเฟกต์โต้ตอบเมื่อผู้ใช้นำเมาส์ไปชี้ (hover) ที่การ์ด เช่น ปุ่ม 'ดูรายละเอียด' เลื่อนขึ้นมา, มีเงา (box-shadow) ที่เด่นชัดขึ้น หรือมีการเคลื่อนไหวอื่นๆ
  • ใช้ CSS Custom Properties (CSS Variables) สำหรับกำหนดค่าสีหลัก, ระยะห่าง หรือรัศมีของขอบ (border-radius) เพื่อให้ง่ายต่อการปรับแต่งธีม
  • ตรวจสอบให้แน่ใจว่าการ์ดมีสไตล์ที่สวยงาม อ่านง่าย และมีระยะห่างที่เหมาะสม ทั้งบนเดสก์ท็อปและมือถือ
