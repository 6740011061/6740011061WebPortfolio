Ex.2 : CSS : Intermediate : สร้างแถบนำทางที่ตอบสนองและมีเมนูดรอปดาวน์

คุณจะได้รับโค้ด HTML สำหรับโครงสร้างแถบนำทางพื้นฐาน รวมถึงรายการเมนูหลักและเมนูดรอปดาวน์บางส่วน งานของคุณคือการใช้ CSS เพื่อทำให้แถบนำทางนี้ดูดี ตอบสนองต่อขนาดหน้าจอที่แตกต่างกัน และมีฟังก์ชันเมนูดรอปดาวน์ที่ทำงานได้อย่างถูกต้อง รวมถึงการจัดการเมนูสำหรับหน้าจอขนาดเล็กโดยไม่ต้องใช้ JavaScript

สิ่งที่โจทย์ต้องการ:

  • การจัดวางแถบนำทาง: จัดเรียงองค์ประกอบใน .navbar (โลโก้, ลิงก์เมนู, ปุ่มสลับ) ให้อยู่ในแนวนอน และจัดกึ่งกลางหรือจัดชิดขอบตามความเหมาะสม
  • การจัดรูปแบบลิงก์: จัดรูปแบบลิงก์เมนูหลักให้สวยงาม เช่น การกำหนดสีพื้นหลัง, สีข้อความ, padding, และการเปลี่ยนสถานะเมื่อวางเมาส์เหนือ (:hover)
  • เมนูดรอปดาวน์ (Desktop): ซ่อน .dropdown-content ไว้แต่แรก เมื่อวางเมาส์เหนือ (:hover) ที่ .dropdown จะต้องแสดง .dropdown-content ขึ้นมา จัดตำแหน่ง .dropdown-content ให้อยู่ใต้ลิงก์หลักอย่างถูกต้องและมีสไตล์ที่ชัดเจน (เช่น มีพื้นหลัง, border-radius, shadow)
  • การตอบสนอง (Responsive Design): ใช้ Media Query เพื่อปรับ layout ของแถบนำทางเมื่อหน้าจอมีขนาดเล็กกว่า 768px (หรือค่าอื่นที่เหมาะสม)
  • เมนู Hamburger (Mobile): บนหน้าจอขนาดเล็ก ให้ซ่อน #menu-toggle-checkbox และ label.menu-toggle บนหน้าจอขนาดใหญ่ และแสดงบนหน้าจอขนาดเล็ก เมื่อ #menu-toggle-checkbox ถูกเลือก (:checked) บนหน้าจอขนาดเล็ก ให้แสดง .nav-links ที่ซ่อนอยู่และซ้อนกันในแนวตั้ง เมนูดรอปดาวน์บนมือถือควรปรับการแสดงผลให้เหมาะสม
